----------------------Темы----------------------

1) async, defer ( html <script> ) !

	1) defer сообщает браузеру, что он должен продолжать обрабатывать страницу и загружать скрипт в фоновом режиме !

	2) async означает, что скрипт абсолютно независим !

2) Массивы !

	1) arr.slice([start], [end]) = Он возвращает новый массив, в который копирует элементы, начиная с индекса start и до end !

	2) arr.indexOf(item, from) = ищет item, начиная с индекса from, и возвращает индекс, на котором был найден искомый элемент, в противном случае -1 !

	3) arr.filter(function(item, index, array) = ищет один (первый попавшийся) элемент, на котором функция-колбэк вернёт true !

	4) arr.reverse = меняет порядок элементов в arr на обратный !

	5) str.split(delim) = он разбивает строку на массив по заданному разделителю delim ! 

	6) arr.join(glue) =  он создаёт строку из элементов arr, вставляя glue между ними !

3) Мемоизация !
	
	0) Мемоизация = сохранение результатов выполнения функций для предотвращения повторных вычислений !

	1) useCallback = Вернет сразу колбэк, если не поменялись, props !

	2) useMemo = Вернет сразу значение, если не поменялись, props !

	3) React.memo = React будет использовать результат последнего рендера, избегая повторного рендеринга !
